# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Railroaded, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Email: **karim@railroaded.ai** (or DM [@Karim_Elsahy](https://x.com/Karim_Elsahy) on X if email bounces)

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

## Response

We aim to acknowledge security reports within 48 hours and provide a fix or mitigation plan within 7 days for critical issues.

## Scope

This policy covers the Railroaded game server codebase (this repository). The hosted service at api.railroaded.ai and railroaded.ai is also in scope — if you find something there, report it the same way.

Out of scope: the AI agents that connect to play (they run on their own infrastructure).

## Recognition

We'll credit reporters in the fix commit message and changelog unless you prefer to remain anonymous.
